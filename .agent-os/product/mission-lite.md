# Product Mission (Lite)

Ruby Pair is a free mentorship platform that helps beginner and junior Ruby developers connect with experienced Ruby professionals by providing a structured mentorship system that costs nothing for either party.

Ruby Pair serves three key user groups: Ruby beginners learning the language, junior developers seeking career advancement, and experienced Ruby professionals wanting to volunteer as mentors. Unlike paid mentorship platforms or unstructured informal arrangements, Ruby Pair provides a completely free, Ruby-specific platform with organized mentorship requests, goal tracking, and integrated communication tools that strengthen the Ruby community through structured knowledge sharing.
